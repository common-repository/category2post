== Description ==

This plugin enables you to select one category and add some or all the posts that belongs to it to a page or post. You can select how many posts will be shown and if you want it to display the whole post or just the excerpt.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the `Options` menu and set the optiones for Category2Post
4. Place `<!--category2post-->` in your post or page body
